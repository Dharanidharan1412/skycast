# Weather Explorer

**Weather Explorer** is a simple web application that allows users to check the current weather and 4-day forecast for any city. It provides temperature, weather description, and icons for a visual representation.

## Features

* Search for weather by city name
* Display current temperature and weather conditions
* 4-day weather forecast
* Responsive and user-friendly interface

## Technologies Used

* **HTML5** – Markup for structure
* **CSS3** – Styling and layout (`styles.css`)
* **JavaScript** – Fetching weather data and updating the DOM (`script.js`)

## How to Use

1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Enter a city name in the search box.
4. Click **Search** to view the current weather and forecast.

## Project Structure

```
Weather-Explorer/
│
├─ index.html        # Main HTML file
├─ styles.css        # CSS styling
└─ script.js         # JavaScript functionality
```
